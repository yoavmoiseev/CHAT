МАТЕМАТИЧЕСКИЕ ФОРМУЛЫ

Формула квадратного уравнения

Квадратное уравнение имеет вид: ax² + bx + c = 0

Решение находится по формуле: x = (-b ± √(b²-4ac)) / 2a

Дискриминант D = b² - 4ac определяет количество корней:

  • D > 0 - два различных корня
  • D = 0 - один корень
  • D < 0 - нет действительных корней

Формула площади

Площадь прямоугольника: S = a × b, где a и b - длины сторон

Площадь круга: S = πr², где r - радиус

Площадь треугольника: S = (a × h) / 2, где a - основание, h - высота

Формула Герона для площади треугольника: S = √(p(p-a)(p-b)(p-c)), где p - полупериметр

Формула производной

Производная функции f(x) в точке x показывает скорость изменения функции.

Обозначение: f'(x) или df/dx

Определение: f'(x) = lim (f(x+h) - f(x)) / h при h→0

Основные правила:

  • (c)' = 0 (производная константы)
  • (x^n)' = n×x^(n-1) (степенная функция)
  • (f+g)' = f' + g' (сумма)
  • (f×g)' = f'×g + f×g' (произведение)

Теорема Пифагора

В прямоугольном треугольнике квадрат гипотенузы равен сумме квадратов катетов.

Формула: a² + b² = c², где c - гипотенуза, a и b - катеты

Логарифм

Логарифм - это степень, в которую нужно возвести основание, чтобы получить данное число.

Обозначение: log_a(b) = c означает a^c = b

Свойства логарифмов:

  • log_a(xy) = log_a(x) + log_a(y)
  • log_a(x/y) = log_a(x) - log_a(y)
  • log_a(x^n) = n×log_a(x)
  • log_a(a) = 1
  • log_a(1) = 0
