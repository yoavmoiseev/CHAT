כתובת IP

כתובת IP (Internet Protocol Address) היא מזהה ייחודי המוקצה למכשיר המחובר לרשת מחשבים. כתובת ה-IP מאפשרת למכשירים לזהות ולתקשר אחד עם השני ברשת.

כתובת IPv4 מורכבת מארבעה מספרים בטווח 0-255, מופרדים בנקודות. לדוגמה: 192.168.1.1

סוגי כתובות IP:
- כתובת IP ציבורית (חיצונית) - משמשת לתקשורת באינטרנט
- כתובת IP פרטית (פנימית) - משמשת ברשתות מקומיות (192.168.x.x, 10.x.x.x)

ישנם שני סוגים עיקריים של כתובות IP:
- IPv4: מורכב מ-32 ביטים (4 בתים). דוגמה: 192.168.0.1
- IPv6: מורכב מ-128 ביטים. דוגמה: 2001:0db8:85a3:0000:0000:8a2e:0370:7334

שרת

שרת הוא מחשב או תוכנה המספקים שירותים למחשבים או תוכנות אחרות (לקוחות) ברשת מחשבים.

סוגי שרתים:
- שרת אינטרנט (Web Server) - מעבד בקשות HTTP ומחזיר דפי אינטרנט
- שרת קבצים (File Server) - מאחסן ומספק גישה לקבצים
- שרת דואר (Mail Server) - מעבד דואר אלקטרוני
- שרת מסדי נתונים (Database Server) - מנהל מסדי נתונים
- שרת משחקים (Game Server) - מאפשר משחקים מרובי משתתפים

השרת פועל על פי מודל לקוח-שרת: הלקוח שולח בקשה, השרת מעבד אותה ומחזיר תשובה.

שרת מקומי (Local Server) הוא שרת הפועל על המחשב שלך ומאפשר גישה רק ברשת המקומית.

פרוטוקול

פרוטוקול הוא מערכת של כללים והסכמים המגדירים את סדר חילופי המידע בין מכשירים ברשת.

דוגמאות לפרוטוקולי תקשורת:
- HTTP/HTTPS - פרוטוקול להעברת היפרטקסט, משמש לדפי אינטרנט
- FTP - פרוטוקול להעברת קבצים
- SMTP - פרוטוקול לשליחת דואר אלקטרוני
- TCP/IP - פרוטוקולי הבסיס של האינטרנט
- UDP - פרוטוקול דאטגרמות משתמש
- WebSocket - פרוטוקול לתקשורת דו-כיוונית בזמן אמת

פרוטוקולים פועלים ברמות שונות של מודל OSI ומבטיחים העברת נתונים אמינה בין מכשירים.

רשת מקומית

רשת מקומית (LAN - Local Area Network) היא רשת מחשבים המכסה אזור גיאוגרפי מוגבל, כמו בית, משרד או בית ספר.

מאפיינים של רשת מקומית:
- מהירות גבוהה (בדרך כלל 100 Mbps עד 1 Gbps ומעלה)
- אזור גיאוגרפי מוגבל
- בעלות ותחזוקה על ידי ארגון אחד
- חיבור באמצעות כבלים (Ethernet) או אלחוטית (Wi-Fi)

שימושים נפוצים:
- שיתוף קבצים בין מחשבים
- שיתוף מדפסות ומכשירים
- משחקים מקומיים מרובי משתתפים
- תקשורת פנימית בארגון

להבדיל מ-WAN (Wide Area Network) שהיא רשת רחבת היקף כמו האינטרנט.

דפדפן

דפדפן (Browser) הוא תוכנה המאפשרת לגלוש באינטרנט ולצפות בדפי אינטרנט.

דפדפנים פופולריים:
- Google Chrome
- Mozilla Firefox
- Microsoft Edge
- Safari (למחשבי Apple)
- Opera

הדפדפן מבצע מספר פעולות:
- שולח בקשות HTTP לשרתים
- מקבל קבצי HTML, CSS, JavaScript
- מעבד ומציג את הדף בצורה ויזואלית
- מנהל עוגיות (Cookies) והיסטוריה
- מספק אבטחה ופרטיות

הדפדפן הוא הממשק העיקרי בין המשתמש לאינטרנט.
