School Auto Chat — Руководство (DEMO)

Это демо-версия. Тестируйте локально и запускайте через run.bat.

Коротко

Обычный чат + TeacherBot — бот, который ищет термины в локальной папке knowledge_base (TXT/HTML). Бот возвращает контекст первого вхождения термина в каждом файле. Техническая рекомендация: термин в базе должен появляться только один раз в файлах (однократное вхождение), чтобы результаты были предсказуемы.

Запуск (Windows)

Запустите:

run.bat

При первом запуске Windows предложит разрешить доступ в брандмауэре. Нажмите Yes / Разрешить — откройте порт для локальной сети, чтобы клиенты могли подключаться.

Ниже — фрагменты из файлов в knowledge_base (демо):

Русский

1.  IT — область технологий, связанная с использованием компьютеров, сетей, программного обеспечения и данных.
31. Firewall — система фильтрации сетевого трафика.

Английский (пример)

1. IT (Information Technology) — Field of technology related to the use of computers, networks, software, and data.
35. Firewall — System that filters network traffic.
38. IP Address (Internet Protocol Address) — Logical address assigned to a network device.

Пояснения

  • Бот ищет точную подстроку запроса (без учёта регистра).
  • Он выдаёт контекст только первого вхождения термина в каждом файле (~300 символов вокруг него).
  • Если точного совпадения нет — будут подсказки похожих слов (автокоррекция).

Рекомендации по оформлению файлов

  • Сохраняйте файлы в UTF-8.
  • Пишите термин один раз в явном виде (например: Firewall — система фильтрации).
  • Разделяйте абзацы и ставьте точки — тогда контекст будет обрезаться корректно по предложениям.
  • Для HTML используйте простые теги (<p>, <h1>) — бот извлекает видимый текст.
