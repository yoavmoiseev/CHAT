School Auto Chat — מדריך (DEMO)

זו גרסת הדמו. יש להריץ מקומית באמצעות run.bat.

סקירה כללית

צ׳אט רגיל עם TeacherBot המשולב שמחפש קבצים מקומיים בתיקייה knowledge_base (TXT/HTML). הבוט מחזיר את ההקשר של המופע הראשון של הכרך בכל קובץ. המלצה: יש להכניס כל מונח פעם אחת בלבד בקבצים כדי לקבל תוצאות צפויות.

הפעלה (Windows)

run.bat

כאשר Windows יבקש לאפשר גישה דרך חומת האש, יש ללחוץ Yes / Allow כדי לפתוח את היציאה לרשת מקומית.

דוגמאות (מה-knowledge_base)

1. IT (טכנולוגיית מידע) — תחום טכנולוגי העוסק בשימוש במחשבים, רשתות, תוכנה ונתונים.
35. Firewall (חומת אש) — מערכת לסינון תעבורת רשת.
38. IP Address (כתובת IP) — כתובת לוגית של התקן ברשת.

אופן החיפוש

  • חיפוש הוא חיפוש תת-מחרוזת ללא תלות באותיות גדולות/קטנות.
  • הבוט מחזיר רק את המופע הראשון בקובץ (כ־300 תווים מסביב).
  • אם אין התאמה מדויקת — מוצגות הצעות למונחים דומים.
