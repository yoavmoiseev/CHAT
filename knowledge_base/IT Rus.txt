# КОМПЬЮТЕР (COMPUTER)

Компьютер (Computer) — это электронное устройство для обработки, хранения и передачи данных.

Основные компоненты:

* CPU — процессор
* RAM — оперативная память
* Storage — накопитель
* Motherboard — материнская плата
* PSU — блок питания

Компьютер работает по принципу: ввод → обработка → вывод → хранение.

---

# МАТЕРИНСКАЯ ПЛАТА (MOTHERBOARD)

Материнская плата (Motherboard) — основная плата компьютера, соединяющая все компоненты.

Основные элементы:

* Socket — разъём для процессора
* Chipset — набор микросхем
* DIMM Slots — слоты для RAM
* PCIe Slots — слоты расширения
* SATA Ports — порты для накопителей
* BIOS/UEFI Chip — микропрограмма загрузки

Материнская плата обеспечивает взаимодействие всех устройств.

---

# ПРОЦЕССОР (CPU – CENTRAL PROCESSING UNIT)

Процессор (CPU) — центральное устройство обработки данных.

Основные характеристики:

* Clock Speed (ГГц) — тактовая частота
* Cores — количество ядер
* Threads — количество потоков
* Cache (L1, L2, L3) — кэш-память
* TDP — тепловыделение

CPU выполняет инструкции программ.

---

# ОПЕРАТИВНАЯ ПАМЯТЬ (RAM – RANDOM ACCESS MEMORY)

RAM — временная память для хранения данных во время работы компьютера.

Типы:

* DDR3
* DDR4
* DDR5

Характеристики:

* Capacity (ГБ) — объём
* Frequency (МГц) — частота
* Latency — задержка

RAM очищается при выключении питания.

---

# НАКОПИТЕЛЬ (STORAGE)

Накопитель (Storage) — устройство для постоянного хранения данных.

Типы:

* HDD (Hard Disk Drive) — жёсткий диск
* SSD (Solid State Drive) — твердотельный накопитель
* NVMe SSD — высокоскоростной SSD через PCIe

Основные параметры:

* Capacity — объём
* Read/Write Speed — скорость чтения/записи
* Interface — SATA / NVMe

---

# БЛОК ПИТАНИЯ (PSU – POWER SUPPLY UNIT)

Блок питания (PSU) — устройство, преобразующее переменный ток в постоянный.

Основные характеристики:

* Wattage (Вт) — мощность
* Efficiency Rating (80 Plus) — эффективность
* Modular / Non-modular — тип подключения кабелей

PSU питает все компоненты компьютера.

---

# ВИДЕОКАРТА (GPU – GRAPHICS PROCESSING UNIT)

Видеокарта (GPU) — устройство для обработки графики.

Типы:

* Integrated GPU — встроенная
* Dedicated GPU — дискретная

Параметры:

* VRAM — видеопамять
* CUDA / Stream Cores — вычислительные ядра
* Clock Speed — частота

GPU используется для игр, видео и вычислений.

---

# СИСТЕМА ОХЛАЖДЕНИЯ (COOLING SYSTEM)

Система охлаждения (Cooling System) — обеспечивает отвод тепла.

Типы:

* Air Cooling — воздушное охлаждение
* Liquid Cooling — жидкостное охлаждение

Компоненты:

* Heatsink — радиатор
* Fan — вентилятор
* Thermal Paste — термопаста

---

# BIOS / UEFI

BIOS (Basic Input Output System) — базовая система ввода-вывода.
UEFI (Unified Extensible Firmware Interface) — современная версия BIOS.

Функции:

* POST (Power-On Self Test)
* Загрузка операционной системы
* Настройка оборудования
* Boot Order — порядок загрузки

---

# КОРПУС (CASE)

Корпус (Case) — физическая оболочка компьютера.

Форм-факторы:

* ATX
* Micro-ATX
* Mini-ITX

Корпус влияет на вентиляцию и совместимость компонентов.

---

# ПОРТЫ И ИНТЕРФЕЙСЫ (PORTS AND INTERFACES)

Порт (Port) — разъём для подключения устройств.

Основные порты:

* USB (Universal Serial Bus)
* HDMI (High-Definition Multimedia Interface)
* DisplayPort
* Ethernet (RJ-45)
* Audio Jack

Интерфейс определяет способ передачи данных.

---

# ОПЕРАЦИОННАЯ СИСТЕМА (OPERATING SYSTEM – OS)

ОС — программное обеспечение, управляющее аппаратной частью компьютера.

Функции:

* Управление памятью
* Управление процессами
* Работа с драйверами
* Управление файловой системой

Примеры:

* Windows
* Linux
* macOS

---

# ДРАЙВЕР (DRIVER)

Драйвер (Driver) — программа, обеспечивающая взаимодействие ОС с оборудованием.

Без драйвера устройство может работать некорректно.

---

# УСТРОЙСТВА ВВОДА (INPUT DEVICES)

Устройства ввода — передают данные в компьютер.

Примеры:

* Keyboard — клавиатура
* Mouse — мышь
* Scanner — сканер
* Microphone — микрофон

---

# УСТРОЙСТВА ВЫВОДА (OUTPUT DEVICES)

Устройства вывода — отображают результат работы.

Примеры:

* Monitor — монитор
* Printer — принтер
* Speakers — колонки

# КОМПЬЮТЕР (COMPUTER)

Компьютер (Computer) — это устройство, которое принимает информацию, обрабатывает её и выводит результат.

Основные части:

* Процессор (CPU) — «мозг»
* Оперативная память (RAM)
* Накопитель (HDD/SSD)
* Материнская плата (Motherboard)
* Блок питания (PSU)

---

# ПРОЦЕССОР (CPU – CENTRAL PROCESSING UNIT)

Процессор (CPU) — главная часть компьютера, которая выполняет вычисления.

Характеристики:

* Частота (GHz) — скорость работы
* Ядра (Cores) — количество вычислительных блоков
* Кэш (Cache) — быстрая внутренняя память

Чем больше ядер и выше частота — тем быстрее работает компьютер.

---

# ОПЕРАТИВНАЯ ПАМЯТЬ (RAM – RANDOM ACCESS MEMORY)

RAM — память, в которой временно хранятся данные во время работы компьютера.

Особенности:

* Измеряется в гигабайтах (GB)
* Очищается при выключении
* Влияет на многозадачность

Если RAM мало — компьютер «тормозит».

---

# ЖЁСТКИЙ ДИСК (HDD – HARD DISK DRIVE)

HDD — устройство для хранения данных.

Особенности:

* Хранит файлы постоянно
* Имеет вращающиеся диски
* Работает медленнее SSD

---

# ТВЕРДОТЕЛЬНЫЙ НАКОПИТЕЛЬ (SSD – SOLID STATE DRIVE)

SSD — современный накопитель без движущихся частей.

Преимущества:

* Быстрее HDD
* Тише
* Надёжнее

Используется для установки операционной системы.

---

# МАТЕРИНСКАЯ ПЛАТА (MOTHERBOARD)

Материнская плата — главная плата компьютера, соединяющая все компоненты.

На ней находятся:

* Разъём для процессора (Socket)
* Слоты для памяти (RAM Slots)
* Разъёмы для накопителей (SATA)
* Разъёмы для видеокарты (PCIe)

---

# ВИДЕОКАРТА (GPU – GRAPHICS PROCESSING UNIT)

Видеокарта — отвечает за изображение на экране.

Типы:

* Встроенная (Integrated)
* Отдельная (Dedicated)

Используется для игр, видео и графики.

---

# БЛОК ПИТАНИЯ (PSU – POWER SUPPLY UNIT)

Блок питания — подаёт электричество всем частям компьютера.

Измеряется в ваттах (W).

Если мощности не хватает — компьютер может не включаться.

---

# МОНИТОР (MONITOR)

Монитор — устройство вывода изображения.

Характеристики:

* Размер (дюймы)
* Разрешение (1920×1080 и т.д.)
* Частота обновления (Hz)

---

# КЛАВИАТУРА (KEYBOARD)

Клавиатура — устройство ввода текста и команд.

---

# МЫШЬ (MOUSE)

Мышь — устройство для управления курсором.

---

# ОПЕРАЦИОННАЯ СИСТЕМА (OPERATING SYSTEM – OS)

ОС — программа, которая управляет компьютером.

Функции:

* Запуск программ
* Управление памятью
* Работа с файлами
* Управление устройствами

Примеры:

* Windows
* Linux

---

# BIOS / UEFI

BIOS / UEFI — программа, которая запускается первой при включении компьютера.

Функции:

* Проверка оборудования (POST)
* Запуск операционной системы
* Настройка порядка загрузки (Boot Order)

---

# ПОРТ (PORT)

Порт — разъём для подключения устройств.

Примеры:

* USB — флешки, мышь, клавиатура
* HDMI — монитор
* Ethernet — интернет-кабель
* Audio Jack — наушники

---

# ДРАЙВЕР (DRIVER)

Драйвер — программа, которая помогает операционной системе работать с оборудованием.

Без драйвера устройство может не работать.

---

Передают информацию в компьютер:

* Клавиатура
* Мышь
* Микрофон
* Сканер

---

# УСТРОЙСТВА ВЫВОДА (OUTPUT DEVICES)

Выводят информацию:

* Монитор

* Колонки

# БЕЗОПАСНОСТЬ ПРИ РАБОТЕ С КОМПЬЮТЕРОМ

Правила:

* Отключать питание перед разборкой
* Не трогать детали мокрыми руками
* Использовать антистатический браслет
* Не перегревать оборудование
