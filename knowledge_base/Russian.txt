Алгебра — раздел математики, использующий буквы и символы для представления чисел и решения уравнений (Algebra).
Алгоритм — последовательность шагов для решения задачи или выполнения действия (Algorithm).
Угол — фигура, образованная двумя лучами, исходящими из одной точки (Angle).
Площадь — количество пространства внутри плоской фигуры (Area).
Атом — наименьшая частица химического элемента, сохраняющая его свойства (Atom).
Среднее значение — результат деления суммы чисел на их количество (Average).
Двоичная система — система счисления, использующая только цифры 0 и 1 (Binary).
Биология — наука о живых организмах (Biology).
Байт — единица цифровой информации, обычно равная 8 битам (Byte).
Клетка — наименьшая единица живого организма (Cell).
Химическая реакция — процесс превращения одних веществ в другие (Chemical reaction).
Хромосома — структура клетки, содержащая генетическую информацию (Chromosome).
Круг — геометрическая фигура, все точки которой равноудалены от центра (Circle).
Климат — долговременный характер погодных условий региона (Climate).
Код — система символов или инструкций, используемая в программировании (Code).
Компьютер — электронное устройство для обработки и хранения данных (Computer).
Конституция — основной закон государства (Constitution).
Континент — один из крупнейших массивов суши Земли (Continent).
Данные — информация, предназначенная для обработки (Data).
База данных — организованная система хранения информации (Database).
Десятичная система — система счисления, основанная на десяти цифрах (Decimal).
Демократия — форма правления, при которой власть принадлежит народу (Democracy).
Плотность — масса вещества в единице объёма (Density).
Производная — скорость изменения функции в математике (Derivative).
Диаметр — линия, проходящая через центр круга и соединяющая две точки окружности (Diameter).
ДНК — молекула, содержащая генетическую информацию организма (DNA).
Экосистема — система взаимодействия живых организмов и среды (Ecosystem).
Электрический ток — поток электрических зарядов (Electric current).
Электричество — форма энергии, связанная с движением заряженных частиц (Electricity).
Элемент — вещество, состоящее из одного типа атомов (Element).
Энергия — способность совершать работу или вызывать изменения (Energy).
Уравнение — математическое равенство двух выражений (Equation).
Испарение — переход жидкости в газообразное состояние (Evaporation).
Эволюция — постепенное изменение живых организмов со временем (Evolution).
Эксперимент — научный опыт для проверки гипотезы (Experiment).
Сила — воздействие, которое может изменить движение тела (Force).
Дробь — число, обозначающее часть целого (Fraction).
Функция — зависимость между входными и выходными значениями (Function).
Галактика — огромная система звёзд, газа и пыли (Galaxy).
Ген — единица наследственности (Gene).
Геометрия — раздел математики, изучающий формы и пространство (Geometry).
Гравитация — сила притяжения между объектами с массой (Gravity).
Аппаратное обеспечение — физические компоненты компьютера (Hardware).
Шестнадцатеричная система — система счисления с основанием 16 (Hexadecimal).
История — наука о событиях прошлого (History).
Гипотеза — предположение, которое можно проверить экспериментом (Hypothesis).
Ввод — данные, вводимые в компьютер (Input).
Целое число — число без дробной части (Integer).
Интернет — глобальная сеть соединённых компьютеров (Internet).
IP-адрес — уникальный числовой идентификатор устройства в сети (IP address).
Информационные технологии — использование компьютеров и сетей для обработки информации (Information Technology).
Клавиатура — устройство для ввода текста в компьютер (Keyboard).
Километр — единица длины, равная 1000 метрам (Kilometer).
Язык — система общения с помощью слов и правил (Language).
Широта — географическая координата север-юг (Latitude).
Долгота — географическая координата восток-запад (Longitude).
Масса — количество вещества в объекте (Mass).
Материя — всё, что имеет массу и занимает пространство (Matter).
Среднее арифметическое — среднее значение набора чисел (Mean).
Память — устройство компьютера для хранения данных (Memory).
Метафора — образное сравнение двух понятий (Metaphor).
Метр — основная единица длины в метрической системе (Meter).
Молекула — группа атомов, соединённых химическими связями (Molecule).
Умножение — математическая операция повторного сложения (Multiplication).
Сеть — система соединённых компьютеров (Network).
Нейрон — нервная клетка, передающая сигналы (Neuron).
Существительное — часть речи, обозначающая предмет или понятие (Noun).
Операционная система — программа, управляющая работой компьютера (Operating system).
Орбита — путь движения объекта вокруг другого объекта (Orbit).
Вывод — данные, полученные в результате обработки (Output).
Кислород — химический элемент, необходимый для дыхания (Oxygen).
Абзац — группа предложений, объединённых одной темой (Paragraph).
Пароль — секретная комбинация для доступа к системе (Password).
Процент — одна сотая часть числа (Percentage).
Периметр — сумма длин всех сторон фигуры (Perimeter).
Фотосинтез — процесс образования питательных веществ у растений с помощью света (Photosynthesis).
Физика — наука о материи, энергии и силах (Physics).
Планета — небесное тело, вращающееся вокруг звезды (Planet).
Многоугольник — фигура с несколькими прямыми сторонами (Polygon).
Население — количество людей в определённой области (Population).
Простое число — число, делящееся только на 1 и на себя (Prime number).
Вероятность — мера возможности наступления события (Probability).
Программирование — создание программ и инструкций для компьютера (Programming).
Оперативная память — память компьютера для временного хранения данных (RAM).
Прямоугольник — четырёхугольник с прямыми углами (Rectangle).
Радиус — расстояние от центра круга до его края (Radius).
Отношение — сравнение двух величин (Ratio).
Наука — систематическое изучение природы и мира (Science).
Сервер — компьютер, предоставляющий данные другим компьютерам (Server).
Программное обеспечение — программы, используемые компьютером (Software).
Солнечная система — Солнце и объекты, вращающиеся вокруг него (Solar system).
Квадрат — фигура с четырьмя равными сторонами и прямыми углами (Square).
Статистика — наука о сборе и анализе данных (Statistics).
Предмет — учебная дисциплина или тема (Subject).
Температура — мера тепла или холода (Temperature).
Теория — научное объяснение явлений (Theory).
Треугольник — фигура с тремя сторонами (Triangle).
Переменная — величина, значение которой может изменяться (Variable).
Скорость — быстрота движения в определённом направлении (Velocity).
Вирус — микроскопический организм или вредоносная компьютерная программа (Virus).
Объём — количество пространства, занимаемого объектом (Volume).
Веб-браузер — программа для просмотра сайтов (Web browser).
